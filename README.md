# Simple-encryption

1*Encrypt_file.py --> encrypt any files using XOR operator with a password.

2*Encrypt_msg.py --> encrypt any msgs, words, texts using the same XOR with a password 

3*Encryp_substi.py -->encrypt a msg by substitution of letters,numbers,symbol(characters)....

1* and 2* has the same goal: encryptions using simple XOR operator....and 
the last one is about substitution of characters
